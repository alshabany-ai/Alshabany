### 🤖 **البوت الأول: بوت استخراج روابط يوتيوب**
> بوت تلجرام احترافي لاستخراج جميع روابط وعناوين الفيديوهات من قوائم التشغيل والقنوات

- 📌 **البوت:** [@YouTube_Playlist_Extractor_bot](https://t.me/YouTube_Playlist_Extractor_bot)
- 📂 **المستودع:** [YouTube-Playlist-Extractor-Bot](https://github.com/Alshabany88/YouTube-Playlist-Extractor-Bot)
- ⭐ **المميزات:** استخراج قوائم التشغيل، جلب فيديوهات القنوات، إرسال ملفات نصية

### 🎬 **البوت الثاني: بوت تحميل صور يوتيوب**
> بوت تلجرام لتحميل صور الفيديوهات (Thumbnails) بأعلى جودة متاحة

- 📌 **البوت:** [@YouTube_Playlist2_bot](https://t.me/YouTube_Playlist2_bot)
- 📂 **المستودع:** [youtube-thumbnail-bot](https://github.com/Alshabany88/youtube-thumbnail-bot)
- ⭐ **المميزات:** تحميل صور بدقة تصل إلى 1080p، دعم الروابط المتعددة، عرض عنوان الفيديو

### 🛠️ **مشاريع قادمة (قريباً)**
- بوت تنزيل من إنستغرام
- بوت ترجمة نصوص
- أداة تحليل بيانات يوتيوب
